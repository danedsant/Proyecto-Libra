# Concepto del proyecto: 
Una Landing Page que muestra el concepto de un producto consistente en una aplicacion similar a "Shazam" para el diagnóstico mecánico

## Problema: 
Altos costos de mantenimiento correctivo y escasez de repuestos en Venezuela

## Tecnologías propuesta: 
  - Transformada Rápida de Fourier (FFT) y Espectrogramas de Mel
  - Modelos de Clasificación de Audio basados en Redes Neuronales Convolucionales 

## Metodología: 
Desarrollo ágil bajo el marco de Innovación y Desarrollo (I+D) y validación con Lean Startup
